Local AI-Powered Security Testing Platform
  • Developed SecurePilot AI, a full-stack local AI-powered security testing platform using React, Node.js, Express.js, and Playwright to automate vulnerability detection, AI-assisted analysis, validation workflow execution, and HTML report generation.
  • Built a multi-layer security scanning pipeline integrating Semgrep, npm audit, and custom heuristic analysis to identify risky patterns including XSS, unsafe rendering, hardcoded secrets, insecure uploads, and authorization issues.
  • Engineered an automated Playwright validation framework that converts vulnerability findings into executable security test suites (.spec.ts) for contextual validation and security workflow automation.
  • Implemented a privacy-focused local AI architecture using Ollama-compatible models such as DeepSeek Coder and CodeLlama, enabling offline vulnerability analysis without exposing source code to external APIs.
  • Designed a built-in validation execution environment and reporting system to run generated Playwright security suites locally, capture execution evidence, and produce downloadable HTML validation reports.
